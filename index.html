About

Enso Studio is a one-person consultancy based in the EU. I help companies own their backend and infrastructure without hiring full-time — Python and Java APIs, cloud infrastructure, CI/CD pipelines, and database systems.

I work as an embedded part of your team, typically 8–16 hours per week, handling the kind of work where you need someone who understands the full stack from application code through to production infrastructure. Happy to be the person your team pings when something is broken and nobody knows why.

Services

API development and maintenance — Flask, FastAPI, Litestar, Django, Spring Boot. Database design and optimization with PostgreSQL and Redis.

Cloud infrastructure and DevOps — AWS (ECS Fargate, VPC, IAM, and more), Terraform, Kubernetes, DigitalOcean. From architecture to day-to-day operations.

CI/CD and delivery — GitLab CI, GitHub Actions. Build, test, and deploy pipelines that your team can rely on.

Cloud migrations — Experience migrating workloads between AWS, Azure, and DigitalOcean in both directions.

Legacy systems — Taking ownership of systems that nobody else wants to touch but everyone depends on.
